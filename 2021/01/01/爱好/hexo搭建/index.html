我一开始语言设置为language: zh-Hans发现不行 然后去next的语言库发现只有zh-CN这个文件 把主站配置语言设置为 language: zh-CN就好了

Next主题的启用

  1. 把主题拷到根目录里的theme文件夹下，主题为文件夹形式。
  2. 打开根目录下的_config.yml
  3. 修改theme的属性，改成主题文件夹名

更改Next主题为中文

  1. 打开根目录下的_config.yml
  2. language属性改成zh-CN

增加标签页和分类页

  1. theme文件夹下的正在使用的主题文件夹
  2. 打开主题的_config.yml（不是根目录下的）
  3. 找到menu属性
  4. 把要启动的注释去掉

主题的修改，直接刷新就行，不需要重新clean,g,s

正确姿势打开标签页和分类页

添加标签页

  1. git命令窗中，输入hexo n page tags
  2. 在根目录里的source文件夹下，会自动生成一个tag的文件夹，里面有一个index.md文件
  3. 打开index.md
  4. 添加上type: "tags"

添加分类页

  1. git命令窗中，输入hexo n page categories
  2. 在根目录里的source文件夹下，会自动生成一个categories的文件夹，里面有一个index.md文件
  3. 打开index.md
  4. 添加上type: "categories"

主题样式的更改

  1. theme文件夹下的正在使用的主题文件夹
  2. 打开主题的_config.yml（不是根目录下的）
  3. 找到scheme属性
  4. 把要启动的注释去掉

更改头像

  1. theme文件夹下的正在使用的主题文件夹
  2. 打开主题的_config.yml（不是根目录下的）
  3. 找到avatar属性
  4. 直接更改avatar的路径

更改标题和作者以及链接

  1. 打开根目录下的_config.yml
  2. 修改title, author等值
  3. 修改url

启用侧边栏社交链接

  1. theme文件夹下的正在使用的主题文件夹
  2. 打开主题的_config.yml（不是根目录下的）
  3. 找到social属性
  4. 把要用的注释去掉

启用友情链接

  1. theme文件夹下的正在使用的主题文件夹
  2. 打开主题的_config.yml（不是根目录下的）
  3. 找到link_title属性
  4. 把要用的注释去掉

新建一篇文章

  1. git命令窗中，输入hexo n filename
  2. 在根目录里的source/_posts文件夹下，会自动生成一个filename.md的文件
  3. 打开filename.md
  4. 添加tags,categories属性

文章里插入图片

安装插件，在git命令窗中，输入npm install hexo-asset-image --save

打开根目录下的_config.yml

找到 post_asset_folder，把这个选项从false改成true

开启打赏功能

  1. theme文件夹下的正在使用的主题文件夹
  2. 打开主题的_config.yml（不是根目录下的）
  3. 找到reward属性
  4. 添加属性，文字，图片

开启订阅公众号

  1. theme文件夹下的正在使用的主题文件夹
  2. 打开主题的_config.yml（不是根目录下的）
  3. 找到wechat_subscriber属性
  4. 添加属性，文字，图片
